# Envious Labs Homebrew Tap

Homebrew tap for [Envious Labs](https://enviouslabs.co) apps.

## EnviousWispr

Free, on-device AI dictation and speech-to-text for macOS. Apple Silicon, macOS 14+.

```sh
brew install --cask saurabhav88/tap/enviouswispr
```

Source and releases: https://github.com/saurabhav88/EnviousWispr
